# Sales Data Analysis

## 📌 Overview

This project focuses on analyzing a sales dataset using **Python**, **Pandas**, and **Matplotlib**. The analysis includes data cleaning, exploratory data analysis (EDA), and data visualization to uncover meaningful business insights and sales trends.

## 📂 Dataset

The dataset contains sales transaction information, including:

* Order details
* Product information
* Sales revenue
* Units sold
* Discounts
* Customer ratings
* Delivery information
* Payment methods

## 🛠️ Tools & Libraries

* Python
* Pandas
* Matplotlib
* Jupyter Notebook

## 📋 Project Workflow

### 1. Data Cleaning

* Checked dataset structure and data types.
* Identified and handled missing values.
* Removed or corrected inconsistent records where appropriate.
* Prepared the dataset for analysis.

### 2. Exploratory Data Analysis (EDA)

Performed analysis to answer business-related questions, such as:

* Revenue distribution
* Sales by city
* Sales by product category
* Seasonal sales performance
* Customer ratings
* Delivery time analysis
* Payment method distribution
* Returned orders analysis

### 3. Data Visualization

Created various visualizations using **Matplotlib**, including:

* Bar charts
* Pie charts
* Histograms
* Box plots
* Line charts
* Stacked bar charts

## 📈 Key Insights

The analysis helps identify:

* Top-performing product categories
* Revenue contribution by different segments
* Seasonal sales trends
* Customer satisfaction patterns
* Delivery performance
* Return behavior
